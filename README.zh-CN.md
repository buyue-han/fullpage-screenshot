# 网页长截图

一款 Chrome 浏览器扩展，支持一键截取完整网页长图、当前可见区域以及自定义区域截图。

## 功能特性

- **整页长截图** – 自动滚动页面并拼接成一张高清完整长图。
- **可见区域截图** – 快速截取当前屏幕显示的内容。
- **区域选择截图** – 在网页上拖动鼠标，精确框选任意区域进行截图。
- **智能拼接** – 截图时自动隐藏固定/悬浮元素（如导航栏），避免重复出现，并在最终图片中还原到正确位置。
- **自动下载** – 截图完成后自动生成带时间戳的 PNG 文件并下载。
- **另存为** – 支持将最近一次截图通过"另存为"对话框重新保存。
- **频率限制处理** – 当触发 Chrome 截图 API 频率限制时，自动递增等待并重试。

## 安装方法

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`。
2. 开启右上角的**开发者模式**。
3. 点击**加载已解压的扩展程序**。
4. 选择项目中的 `fullpage-screenshot` 文件夹。

## 使用说明

1. 点击浏览器工具栏上的扩展图标。
2. 在弹出的面板中选择截图模式：
   - 🗺️ 截取整个网页
   - 👁️ 截取可见区域
   - ✂️ 选择区域截图
3. 截图完成后图片会自动下载到本地。

## 技术栈

- Manifest V3
- 原生 JavaScript
- Canvas API 图片拼接
- Chrome 扩展 API（`tabs`、`scripting`、`downloads`、`captureVisibleTab`）
